# 上海买房地图 · 使用手册

一个帮你和家人研究买房的交互式上海地图。**本手册不涉及代码**，照着做即可。

## 启动

**双击项目里的 `start.command`**，会自动打开浏览器进入地图（网址 `http://localhost:8000`）。

> 首次双击若提示“无法打开，因为来自身份不明的开发者”：在“系统设置 → 隐私与安全性”里点“仍要打开”，之后就能正常双击。

关闭：回到弹出的黑色终端窗口，按 `Control + C`，或直接关闭该窗口。

## 看地图

- **底图**：淡绿背景，显示上海各**行政区**（灰边 + 区名）。
- **三条环线**：珊瑚红虚线圈，由内到外是**内环 / 中环 / 外环**（内环颜色最深）。
- **板块**：白色圆角块（如古北、苏河湾、中山公园、长征），块上有板块名。

## 悬停与点击

- **鼠标移到板块上**：板块亮起**柠檬黄光晕**（聚光效果），表示“正指着它”。
- **点击板块**：右侧弹出该板块详情。**再点一次同一板块**可取消选择、收起详情。
- 选中的板块保持**柠檬黄**高亮。

## 看板块详情（右侧面板）

- **二手成交均价 / 新房均价**：分开显示（某类没有则显示“—”）。
- **优点 / 缺点**：板块的综合评价。
- **小区列表**：每个小区带
  - **类型标签**：绿色“二手” / 橙色“新房”
  - **价格**：二手是成交均价、新房是开盘/备案价
  - **房龄**：二手显示“房龄 X 年”，新房显示“XXXX 交付”

## 新房 / 二手筛选

地图左下角有 **全部 / 只看新房 / 只看二手**：

- 选“只看新房”：**没有新房的板块变灰、不可点**；右侧小区列表只留新房。
- 选“只看二手”：反之。
- 选“全部”：恢复。

## 收藏（★）

- 点开板块，**板块名旁**和**每个小区旁**都有 ★。点一下变橙色 = 收藏，再点取消。
- 收藏的板块在地图上显示**浅黄边框**，方便一眼认出。
- 收藏**自动保存**，关闭重开仍在。
- **收藏的小区会置顶**：在板块详情的小区列表里，收藏过的小区自动排到最前面。

## 多人身份（我 / 闺蜜 …）

- 右侧面板顶部有**身份下拉**。每个人的收藏**互相独立**。
- 切换身份 → 看到的是该身份自己的收藏。
- 点 ✎ 可给当前身份**改名**（比如改成真实昵称）。
- **只看收藏**：勾选后，地图只保留当前身份收藏的板块，其余变灰。

## 发布到云端（第二阶段功能）

点顶部 **「☁ 发布」** 打开发布管理面板，一次性列出所有板块，勾选要公开给家人的板块。

- **打开时默认勾选你收藏的板块**（“我关注的就是要发布的”）。
- **全选 / 全不选 / 勾选所有收藏**：批量操作。“勾选所有收藏”会把发布列表精确设为你当前收藏的板块。
- 发布是**板块级**：发布一个板块，家人就能看到它下面的全部小区（小区的 ★ 只是你的个人关注标记，不影响发布）。
- 现在勾选只是**记录**；第二阶段上线云版后，勾选的板块会发布到网上供家人访问。当前阶段勾不勾都不影响本地使用。

## 更新房产数据（手动）

当前阶段房产数据是**手动维护**的，存放在 `data/housing.json`。想更新某板块的价格、优缺点、小区列表，用文本编辑器打开这个文件按现有格式修改、保存，然后刷新浏览器即可。
（自动抓取功能是后续增强，届时会有专门的更新方式。）

## 跨设备同步（苹果设备，可选）

把整个项目文件夹放进 **iCloud Drive**，其他登录同一 Apple ID 的 Mac 就能同步到同一份收藏。（手机端与非苹果设备的访问是第二阶段云版要解决的。）

## 常见问题

- **打开是空白 / 报错“数据加载失败”**：确认是**双击 start.command 打开**的（不是直接双击 index.html）。地图必须由本地服务提供。
- **改了数据没变化**：刷新浏览器（`Cmd + R`）；若仍不变，强制刷新 `Cmd + Shift + R`。
